# Cara Menjalankan Dashboard pada Streamlit
### Install Environment
> ***conda create --name main-ds python=3.11.5***

> ***conda activate main-ds***

> ***pip install numpy***

> ***pip install pandas***

> ***pip install matplotlib***

> ***pip install seaborn***

> ***pip install streamlit***

### Run Streamlit App

> ***streamlit run tugas.py***
